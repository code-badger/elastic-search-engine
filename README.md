### Elasticsearch StackOverflow Search Engine

Single Page App designed to query StackOverflow articles using,
- Elasticsearch 6.2.4
- ASP.NET Core 2.1
- Angular 6
- NEST - Elastic Client
- Visual Studio Team Service
- Azure App Service
- StackExchange Dumps
